# МастерДом — Лендинг-пейдж для сервиса ремонта бытовой техники

## Описание проекта

Это демонстрационный одностраничный сайт (лендинг-пейдж) для вымышленной компании "МастерДом", которая предоставляет услуги ремонта бытовой техники с выездом на дом. Сайт разработан с использованием современных веб-технологий и демонстрирует профессиональный дизайн с асимметричным макетом, стеклянными эффектами (glass morphism) и плавными анимациями.

### Основные особенности:
- **Адаптивный дизайн**: Сайт корректно отображается на всех устройствах (десктоп, планшет, мобильные).
- **Интерактивные элементы**: Мобильное меню, плавная прокрутка, анимации при скролле.
- **Форма заявки**: Валидация и маска для телефона, демонстрационная отправка формы.
- **Современный UI**: Использование CSS-переменных, backdrop-filter, градиентов и асимметричных элементов.

## Технологии

- **HTML5**: Семантическая разметка
- **CSS3**: 
  - CSS-переменные для тематизации
  - Flexbox и Grid для раскладки
  - Анимации и переходы
  - Glass morphism эффекты
  - Адаптивный дизайн с медиа-запросами
- **Vanilla JavaScript**: 
  - DOM-манипуляции
  - Обработчики событий
  - Валидация формы
  - Маска для телефона

## Инструкция по запуску

1. **Клонируйте или скачайте проект**:
   ```
   # Если проект в репозитории, используйте git clone
   git clone <repository-url>
   cd landing
   ```

2. **Откройте в браузере**:
   - Просто откройте файл `index.html` в любом современном веб-браузере (Chrome, Firefox, Safari, Edge).
   - Для локального сервера (рекомендуется для лучшей работы):
     - Используйте любой локальный сервер, например:
       - Python: `python -m http.server 8000`
       - Node.js: `npx http-server`
       - PHP: `php -S localhost:8000`
     - Откройте `http://localhost:8000` в браузере.

3. **Просмотр сайта**:
   - Сайт полностью функционален без дополнительных зависимостей.
   - Форма заявки работает в демонстрационном режиме (показывает модальное окно успеха).

## Функциональность

### Навигация
- Фиксированная навигационная панель с эффектом при скролле
- Мобильное меню с гамбургер-кнопкой
- Плавная прокрутка к секциям

### Секции сайта
1. **Герой-секция**: Призыв к действию с ключевыми преимуществами
2. **Преимущества**: 4 ключевых преимущества сервиса
3. **Услуги**: Слайдер с основными услугами (холодильники, стиральные машины и т.д.)
4. **Процесс работы**: 4 шага от заявки до гарантии
5. **Статистика**: Ключевые метрики компании
6. **Контакты**: Форма заявки с валидацией

### Интерактивность
- Анимации при появлении элементов при скролле
- Валидация формы в реальном времени
- Маска для ввода телефона
- Демонстрационная отправка формы с модальным окном

## Демо-режим

Проект отмечен как демонстрационный:
- В верхней части сайта есть баннер "Это демонстрационный проект. Все заявки имитируются."
- Форма отправки показывает модальное окно успеха без реальной отправки данных.
